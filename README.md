# Python Patterns

_This is is a repo of python patterns._

## Running the development server

```bash
> cd python-patterns
> virtualenv venv36 -p <path to python36>
> source venv36/bin/activate
> pip install -r requirements.txt
```

## Running the tests

```bash
> tox
```
